Funko Log

Funko Log é um sistema CRUD simples de gerenciamento e exibição de produtos Funko POP!, desenvolvido por diversão como prática de PHP e MySQL com phpMyAdmin.

Tecnologias Utilizadas:

HTML CSS JavaScript PHP SQL

Funcionalidades:

  • Visualização dos Funkos disponíveis no catálogo
  • Administração com cadastro, edição e exclusão de Funkos
  • Sistema feito para uso local ou em hospedagens básicas

Objetivo:

Este projeto foi desenvolvido sem fins comerciais, como forma de entretenimento pessoal e reforço prático de conhecimentos em programação web, especialmente com PHP e integração com banco de dados MySQL. O foco principal foi praticar as operações de CRUD e entender o fluxo de dados em um sistema de loja virtual simples.
