Patrono Neves

Inspirado no legado de Paulo Freire, Patrono da Educação Brasileira, este projeto foi desenvolvido como Trabalho de Conclusão de Curso técnico em Desenvolvimento de Sistemas. O sistema web tem como objetivo democratizar o acesso ao conhecimento por meio de uma plataforma de cursos online gratuitos e acessíveis, com foco em pessoas que buscam qualificação rápida.

Tecnologias Utilizadas:

HTML CSS JavaScript PHP Node.js SQL JSON

Funcionalidades

  • Cadastro de alunos e login
  • Matrícula em cursos (com bloqueio de repetição)
  • Exibição de videoaulas
  • Emissão de certificados com pagamento via sistema
  • Dashboard do aluno com histórico de cursos
  • Solicitação de promoção para professor
  • Cadastro de informações bancárias do professor
  • Dashboard administrativo com gestão de usuários e cursos

Diagramas Utilizados

  • Casos de Uso (UML): mostram os atores principais e suas interações com o sistema
  • DER: define tabelas, chaves primárias e relacionamentos do banco de dados

Público-Alvo

Pessoas fora do sistema educacional formal, interessadas em aprender habilidades práticas de forma rápida e gratuita. A plataforma também contempla aqueles que desejam compartilhar seus conhecimentos de forma simples e acessível.

Motivação

O projeto foi motivado por dados preocupantes da educação brasileira e pela vontade de contribuir com uma solução acessível. A pesquisa de campo revelou uma preferência por conteúdos em vídeo e estudos via celular, o que reforçou o foco na simplicidade e praticidade da plataforma.
