Apolo Veículos

O Apolo Veículos é um sistema web para gerenciamento de negociações de automóveis, criado como projeto acadêmico com foco em boas práticas de desenvolvimento, modelagem de dados e experiência do usuário. Inicialmente chamado de Royal Motors, o sistema evoluiu em escopo e identidade até chegar à versão atual.

Tecnologias Utilizadas:

HTML CSS JavaScript PHP SQL UML

Objetivo

A ideia era desenvolver uma aplicação completa que permitisse o gerenciamento eficiente das negociações de veículos entre clientes, vendedores e administradores, ajudando na organização, no controle de estoque e na automatização dos processos.

Funcionalidades

  • Visualização de veículos disponíveis pelo cliente
  • Formulário para manifestar interesse e iniciar uma negociação
  • Área exclusiva para os vendedores acompanharem e gerenciar as negociações
  • Dashboard para o administrador com controle de estoque de veículos, podendo incluir novos veículos e editar existentes
  • Atualização automática de status: ao marcar um carro como “vendido”, outras negociações com o mesmo carro são canceladas automaticamente e o carro ficará indisponível

Público-Alvo

O sistema foi desenvolvido para a loja fictícia Apolo Veículos, simulando as rotinas e necessidades reais de uma revendedora de automóveis de pequeno a médio porte. A ideia foi construir uma aplicação que permitisse a organização eficiente de negociações, controle de estoque e uma comunicação fácil entre o cliente e o vendedor.

O projeto tem como foco revendas que enfrentam desafios na organização do processo de vendas, especialmente em ambientes onde o volume de negociações pode ser alto e difícil de controlar sem um sistema dedicado.

Motivação do Projeto

A Apolo Veículos ficou amplamente conhecida pela sua especialização na venda de carros populares, tornando-se referência. No entanto, ao tentar expandir seu catálogo para incluir modelos menos procurados, a empresa percebeu que o foco dos clientes continuava nos populares.

Como resultado, as vendas de veículos populares continuaram em grande escala, o que expôs uma dificuldade de gestão devido à alta demanda. A ausência de um sistema gerava problemas no controle de estoque, acompanhamento de negociações simultâneas e na comunicação entre clientes e vendedores.

Assim, o sistema Apolo Veículos foi idealizado como uma solução acadêmica para enfrentar esses desafios, oferecendo um ambiente digital que organiza as negociações, melhora o controle de estoque e proporciona uma experiência mais clara e prática tanto para clientes quanto para a equipe interna da loja.
